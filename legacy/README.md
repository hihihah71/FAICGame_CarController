# Legacy demos

This folder keeps the original standalone OpenCV demos for reference.

- `pointerGameController.py`: moves the mouse pointer with the index finger and clicks with a gesture.
- `leftHolderMouseButton.py`: holds/releases the left mouse button with hand gestures.
- `SteeringWheelDisplay.py`: older steering wheel drawing helper.

The desktop app does not import these files. The current app entry point is `main.py`, with application code in `core/` and `ui/`.
